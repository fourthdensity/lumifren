# Caching & Memory Platforms for Kimi Chat

## Default: Redis
- Fast, in-memory, 30-day persistence
- Monitored at http://localhost:8081

## Alternatives
1. **Qdrant** — Semantic search, vector DB
2. **PostgreSQL + pgvector** — SQL + vector search
3. **Pinecone** — Cloud vector DB
4. **Elasticsearch** — Full-text search
5. **Milvus** — Distributed vector DB

See README.md for switching instructions.
